# 微信视频号 ECH 加密问题解决方案

## 问题
微信视频号使用 ECH (Encrypted Client Hello) 加密技术，导致：
- 无法获取 SNI（域名被加密）
- 无法获取视频 URL
- 无法获取视频标题、预览图等元数据

## 技术说明

### 什么是 ECH？
ECH (Encrypted Client Hello) 是 TLS 1.3 的扩展，用于加密 ClientHello 中的 SNI 字段，防止中间人窥探访问的域名。

### 为什么 WinDivert 无法解密？
WinDivert 工作在网络层，只能看到加密后的数据包，无法解密 TLS 流量。

### 为什么代理模式可以？
代理模式（mitmproxy）作为中间人，可以：
1. 与客户端建立 TLS 连接（使用自签名证书）
2. 与服务器建立 TLS 连接
3. 在中间解密、分析、重新加密流量

## 当前状态
